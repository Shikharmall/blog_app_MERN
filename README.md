# Frontend

## Technologices Used -
   1.React JS
   
   2.Vite
   
   3.Tailwind CSS

## Run Code-
  1.npm install
  
  2.set environment variables
  
  3.npm run dev

## Components-
  1.All Blog(or post) Page(filters, sorting and pagination)
  
  2.Add Blog Page(store image on AWS S3,also you can add tag)
  
  3.View Particular Blog also contain top there viewed blog.
